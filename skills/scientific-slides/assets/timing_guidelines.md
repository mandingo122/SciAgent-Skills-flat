# Presentation Timing Guidelines

## The One-Slide-Per-Minute Rule

**Baseline**: ~1 slide per minute. Adjustments:
- **Complex slides** (data-heavy, detailed figures): 2–3 minutes each
- **Simple slides** (title, section dividers): 15–30 seconds each
- **Key result slides**: 2–4 minutes each
- **Build slides** (animations): Count as multiple slides

### Slide Count by Talk Length

| Duration | Total | Title/Intro | Methods | Results | Discussion | Conclusion |
|----------|-------|-------------|---------|---------|------------|------------|
| 5 min | 5–7 | 1–2 | 0–1 | 2–3 | 1 | 1 |
| 10 min | 10–12 | 2 | 1–2 | 4–5 | 2–3 | 1 |
| 15 min | 15–18 | 2–3 | 2–3 | 6–8 | 3–4 | 1–2 |
| 30 min | 25–30 | 3–4 | 5–6 | 10–12 | 6–8 | 2 |
| 45 min | 35–45 | 4–5 | 8–10 | 15–20 | 8–10 | 2–3 |
| 60 min | 45–60 | 5–6 | 10–12 | 20–25 | 10–12 | 3–4 |

## Time Allocation Examples

### 15-Minute Conference Talk

```
Introduction (2–3 min, 2–3 slides):
├─ Title slide: 30 sec
├─ Hook/Background: 90 sec
└─ Research question: 60 sec

Methods (2–3 min, 2–3 slides):
├─ Study design: 60–90 sec
├─ Key procedures: 60 sec
└─ Analysis: 30–60 sec

Results (6–7 min, 6–8 slides):
├─ Result 1: 2–3 min (2–3 slides)
├─ Result 2: 2 min (2 slides)
└─ Result 3: 2 min (2–3 slides)

Discussion (2–3 min, 3–4 slides):
├─ Interpretation: 60 sec
├─ Prior work: 60 sec
└─ Implications: 60 sec

Conclusion (1 min, 1–2 slides):
├─ Key takeaways: 45 sec
└─ Acknowledgments: 15 sec

Buffer: 1–2 min for transitions
```

### 45-Minute Seminar

```
Introduction (8–10 min, 8–10 slides)
Methods (8–10 min, 8–10 slides)
Results (18–22 min, 16–20 slides)
Discussion (10–12 min, 8–10 slides)
Conclusion (2–3 min, 2–3 slides)
Reserve: 5–10 min for Q&A
```

### Lightning Talk (5 min)

```
Slide 1: Title (15 sec)
Slide 2: The Problem (45 sec)
Slide 3: Your Solution (60 sec)
Slide 4–5: Key Result (2–3 min)
Slide 6–7: Impact + Contact (75 sec)
```

## Timing Per Slide Type

| Slide Type | Time | Strategy |
|-----------|------|----------|
| Title/section divider | 15–30 sec | Brief transition comment |
| Single bullet point | 30–45 sec | Paraphrase + 1–2 sentences |
| Standard bullets (3–4) | 1 min | 15–20 sec per bullet |
| Standard bullets (5–6) | 2 min | Don't read verbatim |
| Equation | 1–2 min | Context → terms → implications |
| Data visualization | 2–3 min | Setup → patterns → finding → stats → transition |
| Multi-panel figure | 2–4 min | Progressive build or panel-by-panel |
| Table | 1–2 min | Guide attention, don't read every cell |

## Pacing Strategies

### Time Checkpoints

**15-minute talk:**
- 3–4 min: Finishing introduction
- 7–8 min: Halfway through results
- 12–13 min: Starting conclusions

**45-minute talk:**
- 10 min: Finishing introduction
- 20 min: Halfway through methods
- 35 min: Finishing results
- 40 min: In discussion

### Running Behind — Recovery

1. Skip backup/secondary slides (prepare these in advance)
2. Summarize instead of detailing
3. Cut discussion, not results
4. **NEVER skip conclusions**

### Running Ahead — Adjustment

1. Expand on key points naturally
2. Provide additional examples
3. Take questions mid-talk (if appropriate)
4. Slow down slightly (don't add filler)

## Practice Schedule

| Talk Type | Practice Runs | Total Time |
|-----------|--------------|------------|
| Lightning (5 min) | 5–7 times | ~3 hours |
| Conference (15 min) | 3–5 times | 4–5 hours |
| Seminar (45 min) | 3–4 times | 6–8 hours |
| Defense (60 min) | 4–6 times | 10–15 hours |

### Practice Progression

1. **Run 1 — Rough draft**: Get through all slides, time it (will run long), identify problems
2. **Run 2 — Smoothing**: Improve transitions, practice wording, time each section
3. **Run 3 — Refinement**: Exact timing with visible timer, implement strategies
4. **Run 4 — Final polish**: Record yourself, practice Q&A, perfect timing
5. **Run 5+ — Maintenance**: Day before, morning of

### Practice Methods

- **Solo**: Full talk with timer, speak aloud, stand and gesture
- **Recorded**: Video yourself, watch for filler words ("um", "uh"), pace variations
- **Live audience**: Lab meeting, request honest feedback, note confusing sections

## Handling Time Constraints

**Time cut short** ("Can you do 10 instead of 15?"): Keep intro (brief), mention methods (30 sec), show main result only (3 min), brief conclusion. Pre-prepare: know "must keep" vs "optional" slides.

**Extra time given**: Go deeper on results, show backup slides, extend discussion. Don't: repeat content, add filler, slow artificially.

**Q&A within your slot**: Reserve 20–30% for questions. Finish content 2–3 min early. Keep answers to 30–90 sec.
