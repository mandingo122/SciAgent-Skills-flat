## Advanced Alignment

### Pairwise Alignment Parameters

```python
from skbio.alignment import local_pairwise_align_ssw, global_pairwise_align_nucleotide

# StripedSmithWaterman for local alignment (fastest)
alignment, score, start_end = local_pairwise_align_ssw(
    seq1, seq2,
    # SSW parameters set at object level:
    # match_score=2, mismatch_score=-3
)
print(f"Score: {score}, Target: {start_end[0]}, Query: {start_end[1]}")

# Global alignment with custom gap penalties
alignment, score, start_end = global_pairwise_align_nucleotide(
    seq1, seq2,
    match_score=1.0,
    mismatch_score=-2.0,
    gap_open_penalty=5.0,
    gap_extend_penalty=2.0,
)

# Protein alignment with substitution matrix
from skbio.alignment import local_pairwise_align_protein
alignment, score, _ = local_pairwise_align_protein(prot1, prot2)
```

### TabularMSA Operations

```python
from skbio.alignment import TabularMSA
from skbio import DNA

msa = TabularMSA.read('alignment.fasta', constructor=DNA)

# MSA analysis
consensus = msa.consensus()
conservation = msa.conservation()  # per-position conservation scores
entropies = msa.position_entropies()  # Shannon entropy per position
gc_content = msa.gc_content()  # per-sequence GC

# Slicing and filtering
subset = msa[0:10]           # first 10 sequences
column = msa[:, 5:15]       # columns 5-14
no_gaps = msa.gap_frequencies(axis='position') < 0.5
filtered_cols = msa[:, no_gaps]

# Reassign IDs
msa.reassign_index(mapping={'old_id': 'new_id'})
```

## Tree Construction Algorithms

| Algorithm | Function | Complexity | Best For |
|-----------|----------|-----------|----------|
| Neighbor Joining | `nj(dm)` | O(n³) | General purpose, <1000 taxa |
| UPGMA | `upgma(dm)` | O(n²) | Assumes molecular clock |
| WPGMA | `wpgma(dm)` | O(n²) | Weighted version of UPGMA |
| Greedy Min Evolution | `gme(dm)` | O(n² log n) | Large trees, >1000 taxa |
| Balanced Min Evolution | `bme(dm)` | O(n² log n) | Large trees, balanced topology |

```python
from skbio.tree import nj, upgma, gme, bme

# Compare tree construction methods
nj_tree = nj(dm)
upgma_tree = upgma(dm)
gme_tree = gme(dm)

# Robinson-Foulds distance between methods
print(f"NJ vs UPGMA RF: {nj_tree.compare_rfd(upgma_tree)}")
print(f"NJ vs GME RF: {nj_tree.compare_rfd(gme_tree)}")
```

### Advanced Tree Operations

```python
from skbio import TreeNode

tree = TreeNode.read('tree.nwk')

# Rerooting
tree.root_at_midpoint()
tree.root_at('taxon_name')

# Traversal
for node in tree.preorder():    # root → leaves
    print(node.name, node.length)
for node in tree.postorder():   # leaves → root
    pass
for node in tree.levelorder():  # breadth-first
    pass

# Tree metrics
total_length = tree.descending_branch_length()
depths = tree.depths()  # {node: depth} dict
max_distance = tree.maxdist()  # diameter of tree

# Bifurcation and caching
tree.bifurcate()  # resolve polytomies
tree.cache_attr(lambda n: len(list(n.tips())), 'ntips')
```

## BIOM Table Operations

```python
from skbio import Table
import numpy as np

# Create table from arrays
data = np.array([[10, 20, 30], [5, 15, 25], [8, 12, 18]])
table = Table(data,
              observation_ids=['OTU1', 'OTU2', 'OTU3'],
              sample_ids=['S1', 'S2', 'S3'])

# Filtering
filtered = table.filter(
    lambda row, id_, md: row.sum() > 10, axis='observation')
filtered = table.filter(
    lambda col, id_, md: col.sum() > 20, axis='sample')

# Normalization
relative = table.norm(axis='sample', inplace=False)
# CSS normalization (cumulative sum scaling)

# Transform
log_table = table.transform(lambda data, id_, md: np.log1p(data))

# Merge tables
merged = table.merge(other_table)

# Metadata
table.add_metadata({'S1': {'group': 'control'}}, axis='sample')

# Export formats
df = table.to_dataframe()                    # pandas
table.to_hdf5('output.biom')                # HDF5
json_str = table.to_json('Generated by X')  # JSON
```

## Protein Embeddings

```python
from skbio.embedding import ProteinEmbedding, ProteinVector
import numpy as np

# Create embedding from language model output
# embedding_array shape: (sequence_length, embedding_dim)
embedding = ProteinEmbedding(
    embedding_array,
    sequence="MKTLLILAVVAAALA",
    sequence_ids=["protein1"]
)

# Fixed-length vector representation
vector = ProteinVector(
    np.mean(embedding_array, axis=0),  # mean pooling
    sequence="MKTLLILAVVAAALA"
)

# Convert to distance matrix for downstream analysis
dm = embedding.to_distances(metric='euclidean')

# Ordination visualization
pcoa_results = embedding.to_ordination(metric='euclidean', method='pcoa')

# Export for ML
array = embedding.to_array()
df = embedding.to_dataframe()
```

## Integration Patterns

### With QIIME 2

```python
# QIIME 2 artifacts use scikit-bio objects internally
# Export QIIME 2 artifact → scikit-bio
# qiime tools export --input-path table.qza --output-path exported/
from skbio import Table, TreeNode, DistanceMatrix

table = Table.read('exported/feature-table.biom')
tree = TreeNode.read('exported/tree.nwk')
dm = DistanceMatrix.read('exported/distance-matrix.tsv')
```

### With pandas/matplotlib

```python
import pandas as pd
import matplotlib.pyplot as plt
from skbio.stats.ordination import pcoa

# PCoA plot with metadata
pcoa_results = pcoa(dm)
coords = pcoa_results.samples
metadata = pd.read_csv('metadata.tsv', sep='\t', index_col=0)
merged = coords.join(metadata)

fig, ax = plt.subplots(figsize=(8, 6))
for group in merged['treatment'].unique():
    mask = merged['treatment'] == group
    ax.scatter(merged.loc[mask, 'PC1'], merged.loc[mask, 'PC2'], label=group)
ax.set_xlabel(f"PC1 ({pcoa_results.proportion_explained.iloc[0]:.1%})")
ax.set_ylabel(f"PC2 ({pcoa_results.proportion_explained.iloc[1]:.1%})")
ax.legend()
plt.tight_layout()
```

## Performance Tips

| Scenario | Recommendation |
|----------|---------------|
| Large FASTA files (>1M seqs) | Use generator: `skbio.io.read(path, format='fasta', constructor=DNA)` |
| Large trees (>1000 taxa) | Use GME/BME instead of NJ |
| Large beta diversity matrix | Use `partial_beta_diversity()` for specific pairs |
| BIOM tables | Use HDF5 format (`.to_hdf5()`) over JSON |
| Repeated distance lookups | Cache `dm.data` as numpy array |
| Memory-bound operations | Process sequences in batches with generator |
